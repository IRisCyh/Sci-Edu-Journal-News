# Education Journal News

Education Journal News is a local reading dashboard for monitoring science education, STEM education, educational technology, and learning sciences journal updates.

Open the live reading page here:

https://iriscyh.github.io/Sci-Edu-Journal-News/

## Files

- `index.html`: browser-friendly reading window.
- `science-education-journal-digest.html`: same reading window under the automation output filename.
- `science-education-journal-digest.md`: Markdown log used as the source record.

## Current Scope

The digest tracks 11 journals in a fixed priority order and keeps DOI links, English abstract notes, readable Chinese summaries, and daily push records.
